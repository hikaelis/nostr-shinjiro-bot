# 小泉進次郎bot on Nostr

Nostr(とX(Twitter))上で小泉進次郎氏風の投稿するBotです。  
発言内容は、GPT-4のAPIを用いて生成しています。

## 開発環境

- node v18.17.1
- npm v9.6.7
- GitHub Actions(定期実行用)
